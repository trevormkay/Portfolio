The Solution

FROM CIRRUS TO STRATUS

A suite of web apps was developed for a cloud OS platform. These included specialised interfaces for all your media. Because everything was in the cloud, you had no need for local storage, just stream it all.

The Process

FORMING A WHOLE SYSTEM

The seed of this project was a UI library. Creating a set of themes from the start was important in insuring consistency throughout the app.

Interactive prototypes played an integral roll in the design of these interfaces. It allowed me to show the developers exactly how I wanted interactions to perform.

Icons were required throughout the interfaces which led me to creating a suite of 140 icons. This suite included multiple resolutions for use within the apps themselves as well as marketing.
